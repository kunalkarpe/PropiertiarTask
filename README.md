#  Property Listing Application

##  Overview
This is a property listing application built using Next.js. It allows users to filter properties based on location, type (Buy/Rent), and BHK configuration. The UI is responsive and optimized for performance.

---

##  Features
- Filter properties using URL search params
- Dynamic UI with responsive design
- Optimized image loading with Next.js Image
- Skeleton loaders for better UX
- Clean and scalable component structure

---

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- Lucide Icons
